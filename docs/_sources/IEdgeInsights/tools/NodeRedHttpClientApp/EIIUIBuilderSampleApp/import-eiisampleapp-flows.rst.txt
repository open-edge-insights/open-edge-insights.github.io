
Import EIISampleAPP Flows
-------------------------


* 
  Import the Workflow `flows.json <https://github.com/open-edge-insights/eii-tools/blob/master/NodeRedHttpClientApp/EIIUIBuilderSampleApp/flows.json>`_ file to NodeRed dashboard using ``menu`` icon in top right corner as follows

  .. code-block::

     ![images/imageimportnodes.png](https://raw.githubusercontent.com/open-edge-insights/eii-tools/master/NodeRedHttpClientApp/EIIUIBuilderSampleApp/images/imageimportnodes.png)

* 
  Click ``Import`` Button

* 
  Update ``url`` in all nodes wherever applicable and use ``https`` if you are running RDE in ProdMode.
