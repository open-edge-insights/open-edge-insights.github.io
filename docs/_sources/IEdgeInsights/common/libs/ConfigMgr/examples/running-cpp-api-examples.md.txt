## Running Cpp API examples

1. Running the Publisher example:

    ```sh
        $ ./pub
    ```

2. Running the Subscriber example:

    ```sh
        $ ./sub
    ```

3. Running the Server example:

    ```sh
        $ ./server
    ```

4. Running the Client example:

    ```sh
        $ ./client
    ```

5. Running the GetConfig & watchCallback example:

    ```sh
        $ ./sample_app
    ```
