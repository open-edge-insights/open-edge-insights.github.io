
Accessing Web Visualizer and EtcdUI
-----------------------------------

  Environment EtcdUI & WebVisualizer will be running in Following ports. 


* EtcdUI

  * ``https://master-nodeip:30010/``

* WebVisualizer

  * PROD Mode --  ``https://master-nodeip:30007/``
  * DEV Mode -- ``http://master-nodeip:30009/``
